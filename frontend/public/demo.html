US5: Results Dashboard and CSV/JSON Export - API Demo

This demonstrates the backend results endpoints working correctly.

1. GET /api/experiments/{experimentId}/results

Returns summary of all participant sessions including timestamps and duration.


    

    
    

        
2. POST /api/experiments/{experimentId}/results/export (CSV)

        
Exports participant data as CSV with aggregated metrics per participant.

        
        

    

    
    

        
3. POST /api/experiments/{experimentId}/results/export (JSON)

        
Exports full interaction details for each participant in JSON format.
